# 🏦 FinShield AI

## Intelligent Banking Risk & Customer Insights Portal

FinShield AI is a Python-based BFSI analytics application built using Streamlit.

## Features

- Customer Financial Assessment
- Financial Health Score
- Loan Eligibility Analysis
- Risk Classification
- AI Relationship Manager Recommendations
- Interactive Dashboard
- PDF Report Generation

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- ReportLab

## Project Structure

```
FinShield-AI
│
├── app.py
├── financial_engine.py
├── report_generator.py
├── requirements.txt
├── README.md
├── assets/
└── reports/
```

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
